# Asteroids

A classic Asteroids arcade game clone built with vanilla JavaScript and HTML5 Canvas. Play it directly in your browser — no dependencies, no build step.

## Play

Visit the GitHub Pages site for this repo, or open `index.html` locally.

## Controls

### Keyboard
| Key | Action |
|-----|--------|
| **W** / **↑** | Thrust forward |
| **A** / **←** | Rotate left |
| **D** / **→** | Rotate right |
| **Space** | Shoot |
| **Enter** | Start / Restart |

### Mobile
On-screen touch buttons are displayed automatically on touch devices.

## Features

- Vector-art glowing aesthetic on a dark space background
- Asteroids split into smaller pieces when shot
- Score tracking and 3-life system
- Increasing difficulty each level (more and faster asteroids)
- Screen-wrap for ship, bullets, and asteroids
- Responsive canvas that fills the viewport
- Mobile touch controls
